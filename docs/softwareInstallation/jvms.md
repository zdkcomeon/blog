# Jvms 
::: tip 前言
git可以管理项目版本，nvm可以管理node版本，对应到jdk中有 **Jvms（Java version manage system）** 管理Jdk的版本，Jvms会自动管理系统的Jdk版本，安装指定的JDK版本，同时可以自由切换系统JDK版本，环境变量也会自动切换。<br/>
**题外话** <br/>
现在的IDE都一般比较智能，打开一个项目，配置以下jdk的安装路径即可，不需要配置什么环境变量，个人倾向于这种方式。
:::
